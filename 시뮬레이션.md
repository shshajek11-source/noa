# 캐릭터 시뮬레이터 페이지 설계서

## 한 줄 요약
> **"이 아이템 끼면 내 캐릭터 능력치가 얼마나 오를까?"** 를 미리 볼 수 있는 페이지

---

## 1. 이 페이지가 하는 일

```
사용자가 하는 것:
1. 내 캐릭터 검색해서 등록
2. 오른쪽에서 장비/스킬/아르카나 클릭해서 변경
3. 하단에서 "뭐가 얼마나 오르는지" 확인

결과:
→ 실제로 아이템 사기 전에 미리 시뮬레이션 가능
→ 돈 낭비 방지!
```

## 3. 주요 기능 설명

### 3-1. 캐릭터 등록

```
동작 방식:
1. 검색창에 캐릭터 이름 입력 (예: "도룡뇽")
2. 검색 결과에서 내 캐릭터 선택
3. 왼쪽 + 오른쪽에 동일한 캐릭터 정보가 표시됨

데이터 출처:
- 기존 상단 검색 기능과 동일한 방식 사용
- 캐릭터 상세 API에서 장비/스킬/아르카나 정보 가져옴
```

### 3-2. 장비 변경

```
동작 방식:
1. 오른쪽 시뮬레이션 캐릭터의 장비 슬롯 클릭 (예: 무기)
2. 팝업창이 뜸 → 해당 부위 아이템 목록 표시
3. 원하는 아이템 선택
4. 시뮬레이션 캐릭터에 적용됨
5. 하단 변화량이 자동으로 업데이트

중요:
- 무기 슬롯 클릭 → 무기만 나옴 (다른 부위 안 나옴)
- 투구 슬롯 클릭 → 투구만 나옴
- 등급/직업으로 필터링 가능
```

### 3-3. 스킬 레벨 변경

```
동작 방식:
1. 오른쪽 시뮬레이션 캐릭터의 스킬 클릭
2. 슬라이더 또는 +/- 버튼으로 레벨 조절
3. 변화량 자동 업데이트

범위:
- 현재 레벨 ~ 최대 레벨 사이에서 조절
- 다른 스킬로 교체는 불가 (레벨만 변경)
```

### 3-4. 강화/돌파 시뮬레이션

```
동작 방식:
1. 장비 선택 후 강화 단계 조절
2. 예: +10강 → +15강으로 변경
3. 템렙 변화량 자동 계산

템렙 증가 공식 (확인된 정보):
┌─────────────────┬──────────────┐
│ 강화 단계       │ 템렙 증가    │
├─────────────────┼──────────────┤
│ +1강 ~ +9강     │ 강당 +1      │
│ +10강 ~ +15강   │ 강당 +2      │
└─────────────────┴──────────────┘

예시:
- 0강 → 10강 = 템렙 +10
- 10강 → 15강 = 템렙 +10 (강당 +2 × 5)
- 0강 → 15강 = 템렙 +20

⚠️ 주의:
- 공격력/방어력 구체적 증가량은 공개된 자료 없음
- 템렙 기준으로만 계산 (정확한 전투력은 참고용)
```

### 3-5. 변화량 표시

```
표시 방식:
┌─────────────────────────────────────┐
│ 공격력    +800  ⬆️  (초록색)        │
│ 방어력    +200  ⬆️  (초록색)        │
│ 치명타    -50   ⬇️  (빨간색)        │
│ 명중      +0        (회색)          │
└─────────────────────────────────────┘

계산 방식:
- 아이템 스탯 차이를 단순 비교
- 현재 아이템 스탯 vs 시뮬레이션 아이템 스탯
```

## 4. 추가 기능 (2단계)

### 4-1. 다른 유저 장비 복사

```
사용 시나리오:
"랭킹 1등은 뭐 끼고 있지? 나도 똑같이 맞추면 어떨까?"

동작 방식:
1. [장비 불러오기] 버튼 클릭
2. 다른 캐릭터 이름 검색
3. 해당 캐릭터의 장비를 시뮬레이션에 적용
4. 내 캐릭터와 비교
```

### 4-2. 빌드 공유

```
사용 시나리오:
"이렇게 맞추려는데 어때?" 친구한테 공유하고 싶을 때

동작 방식:
1. 시뮬레이션 완료 후 [링크 복사] 클릭
2. URL이 클립보드에 복사됨
3. 친구에게 전송
4. 친구가 링크 열면 동일한 시뮬레이션 결과 확인
```

### 4-3. 목표 설정 (추천 기능)

```
사용 시나리오:
"템렙 100 찍으려면 뭘 바꿔야 해?"

동작 방식:
1. 목표 템렙 입력 (예: 100)
2. 시스템이 추천 방법 제시:
   - "무기 +15강 → +20강 하면 템렙 +10"
   - "흉갑을 에픽으로 바꾸면 템렙 +5"
   - "반지 강화하면 템렙 +3"

⚠️ 제한사항:
- 현재는 템렙 기준으로만 추천 가능
- 전투력 기준 추천은 공식 부족으로 어려움
```

## 6. 제한사항 및 안내 문구

### 화면에 표시할 안내

```
⚠️ 안내사항

1. 전투력/스탯 변화량은 참고용입니다.
   - 실제 게임 내 수치와 다를 수 있습니다.

2. 강화 시뮬레이션은 템렙 기준입니다.
   - 공격력/방어력 정확한 증가량은 제공하지 않습니다.

3. 버프, 파티 시너지 등은 반영되지 않습니다.
```

## 9. 질문 & 답변

**Q: 강화하면 능력치가 정확히 얼마나 오르나요?**
> A: 아쉽게도 공개된 자료가 없습니다. 템렙 증가량만 알 수 있어서, 템렙 기준으로 대략적인 변화만 보여드립니다.

**Q: 실제 전투력이랑 다르면 어떡하나요?**
> A: "참고용"이라고 안내합니다. 정확한 전투력은 게임 내에서만 확인 가능합니다.

**Q: 아이템 변경하면 해당 부위만 나오나요?**
> A: 네! 무기 클릭하면 무기만, 투구 클릭하면 투구만 나옵니다.

### 10-9. 피해야 할 디자인 요소

| 피해야 할 것 | 이유 | 대안 |
|-------------|------|------|
| 둥근 모서리 (border-radius: 20px) | 너무 부드러움, 게임과 안 맞음 | 각진 모서리 또는 clip-path |
| 밝은 배경색 | 게임 UI는 어두움 | #0a0a0f ~ #1a1a25 |
| 그림자 (box-shadow) 남발 | AI스러움 | 글로우(발광) 효과로 대체 |
| 파란색 링크 | 웹사이트 느낌 | 골드/주황 계열 |
| 큰 여백 | 심심해 보임 | 장식선, 테두리로 채움 |
| 이모지 텍스트 | 유치해 보임 | 게임 스타일 아이콘 |
