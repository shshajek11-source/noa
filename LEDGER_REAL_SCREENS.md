## 📋 주요 컴포넌트 명칭 정리

### 1. 헤더
- **컴포넌트**: `header` (styles.header)
- **내용**:
  - 💰 가계부 (제목)
  - 캐릭터별 수입을 관리하고 추적하세요 (부제목)

### 2. CompactKinaOverview
- **표시 조건**: `activeTab !== 'dashboard'` (캐릭터 선택 시에만 표시)
- **레이아웃**: 2열 그리드
- **섹션**:
  - 📊 일일 수입 (todayContentIncome + todayItemIncome)
  - 📈 이번주 수입 (weeklyContentIncome + weeklyItemIncome)

### 3. LedgerTabs
- **컴포넌트**: `LedgerTabs`
- **탭 종류**:
  - 총합 (dashboard) - 📊 아이콘
  - 캐릭터 탭들 (동적 생성)
  - 캐릭터 추가 버튼 - ➕ 아이콘

### 4. FloatingDateButton
- **컴포넌트**: `FloatingDateButton`
- **위치**: 우하단 플로팅
- **기능**: 날짜 선택 모달 열기

### 5. DashboardSummary (총합 탭)
- **컴포넌트**: `DashboardSummary`
- **섹션**:
  1. 📈 전체 수입 현황
     - 오늘 총 수입
     - 이번주 총 수입
  2. 👥 캐릭터별 수입
     - 캐릭터 카드 그리드
  3. 📦 미판매 아이템
     - 등급별 미판매 개수

### 6. PremiumContentSection
- **컴포넌트**: `PremiumContentSection`
- **카드**: `PremiumContentCard`
- **내용**:
  - 슈고페스타 🎪
  - 어비스회랑 🏛️
- **기능**:
  - 이미지 카드 형태
  - 진행 표시 (●●○○○○)
  - ➖ / ➕ 버튼으로 완료 횟수 조절

### 7. ContentIncomeSection
- **컴포넌트**: `ContentIncomeSection`
- **행**: `ContentIncomeRow`
- **내용**:
  - 초월의 길 ⚔️ (transcend)
  - 대원정 🧭 (expedition)
  - 성역 🛡️ (sanctuary)
- **입력 항목**:
  - 남은 횟수 입력
  - 단계 선택 드롭다운
  - ➖ / ➕ 완료 횟수 버튼
  - ○ 2배 토글 버튼
- **하단**: 오늘 컨텐츠 총합

### 8. DailyContentSection
- **컴포넌트**: `DailyContentSection`
- **카드**: `DailyContentCard`
- **내용**:
  - 일일 퀘스트 ⚡
  - 화염의 신전 🔥
  - 얼음 동굴 ❄️
- **기능**:
  - 원형 진행바 (CircularProgress)
  - ➖ / ➕ 버튼

### 9. WeeklyChart
- **컴포넌트**: `WeeklyChart`
- **내용**:
  - 최근 7일 막대 그래프
  - ▓ 컨텐츠 수입 (진한 막대)
  - ░ 아이템 수입 (연한 막대)
  - 총 수입 / 일평균 표시

### 10. ItemManagementTab
- **컴포넌트**: `ItemManagementTab`
- **구성**:
  1. FavoriteItemsPanel (즐겨찾기 패널)
     - 즐겨찾기 아이템 카드 (빠른 추가 버튼)
  2. 보유 아이템 목록
     - 필터 (전체/미판매/판매완료)
     - 등급 필터
  3. EnhancedItemCard (아이템 카드)
     - 등급 색상 표시
     - 상태 (🔴 미판매 / 🟢 판매 완료)
     - 버튼 (판매 완료 / 즐겨찾기 / 삭제)
  4. 아이템 요약
